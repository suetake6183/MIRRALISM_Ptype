# Title: 自動クリーンアップシステム実装
# Status: pending
# Dependencies: 1
# Priority: medium
# Description: 一時ファイルの自動管理と整理を行うシステムを実装する
# Details:
一時ファイル管理戦略に基づき、以下の機能を実装する：

1. 起動時に実行される自動クリーンアップスクリプト（cleanup.py）を作成
   - 7日以上前の/daily/ディレクトリを/archive/へ移動
   - 30日以上前の/archive/ディレクトリを削除
   - ルートディレクトリの「迷子ファイル」を/tmp/へ強制移動

2. 一時ファイル作成ユーティリティ関数を実装
   - 必須プレフィックス（test_、memo_、draft_、temp_）を強制
   - 日付フォルダの自動生成（/tmp/daily/YYYY-MM-DD/）
   - ファイル名に日付を自動付与

3. 定期実行のためのスケジューラー設定
   - システム起動時に自動実行
   - 毎日深夜に自動実行（バックアップ用）

使用技術：
- Python 3.10以上
- pathlib（パス操作）
- shutil（ファイル操作）
- datetime（日付処理）
- schedule 1.2.0（定期実行）

# Test Strategy:
1. 各クリーンアップ機能の単体テスト
   - 古いファイルの移動テスト
   - 古いアーカイブの削除テスト
   - 迷子ファイルの移動テスト
2. 日付フォルダ自動生成のテスト
3. ファイル命名規則の強制テスト
4. エッジケース（権限エラー、ファイルロック等）のテスト
